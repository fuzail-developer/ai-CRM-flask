AI-Powered CRM System
This repository contains a modern Customer Relationship Management (CRM) application built with Python and Flask. It is designed to demonstrate how AI agents can be integrated into business workflows to automate customer data management and provide actionable insights.

🚀 Features
Full CRUD Operations: Easily create, view, update, and delete customer records.

AI Agent Integration: Built-in support for agentic workflows to handle automated tasks.

User Authentication: Secure login and signup system for data protection.

Responsive Dashboard: A clean and user-friendly interface for managing client data.

Database Management: Uses SQLAlchemy for robust data handling and persistence.

🛠️ Tech Stack
Backend: Python, Flask

Database: SQLite / SQLAlchemy

Frontend: HTML5, CSS3, Jinja2 Templates

AI Tools: Integrated with LangChain/OpenAI (Agentic Framework)

📋 Prerequisites
Before running the project, ensure you have:

Python 3.8 or higher installed.

An active GitHub account for version control.
